## Reproduction Steps:

Assumes MongoDB is installed and running in the local environment.

1. Install repo dependencies:
  * $ npm install

2. Seed the test data:
  * $ npm run mongo-seed

3. Start the local server:
  * $ npm run mongo-start

4. Routs:
  * http://localhost:4060/
